## Detailed Description

**SEO Booster — The SEO Toolbox in Your Browser**

Instantly see and fix SEO issues on any page. 4 tools in one popup, zero bloat.

**🛠️ Tools Included**

🏷️ **Meta Tag Viewer** — See all meta tags, Open Graph tags, and Twitter cards at a glance. Color-coded: green = present, yellow = recommended, red = missing critical.

📱 **OG Preview** — See exactly how your page looks when shared on Facebook, Twitter, and LinkedIn. Visual card preview with title, description, and image.

📑 **Heading Structure** — View the complete H1-H6 hierarchy of any page. Spot missing H1s, wrong nesting, or bloated heading trees.

✏️ **Live Edit** — Modify title, description, og:title, og:description, and og:image right in the popup. Changes reflect instantly on the page — perfect for testing before committing.

**🛡️ Privacy First**

- No analytics, no tracking, no data collection
- Only reads metadata from the current tab
- Everything runs locally in your browser
- Works on any website

**📋 How to Use**

1. Install SEO Booster
2. Navigate to any page
3. Click the extension icon
4. Browse tabs: Meta Tags → OG Preview → Headings → Edit

**Why SEO Booster?**

Meta SEO Inspector has 300K+ users but only shows info — it can't edit or preview. SEO Booster gives you view + preview + edit in one free tool.

**Free forever. No ads. No tracking. Open source.**
